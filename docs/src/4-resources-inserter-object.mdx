# InserterObject

InserterObjects are used to generate a UI menu for adding shards.

#### Object shape
```jsx
{
  label: "Markdown",
  type: "markdown"
}
```

#### Object properties
| Name | Type    | Description                                                            |
|------|---------|------------------------------------------------------------------------|
| label | string | Button text for letting users know what they're inserting.             |
| type  | string | Type of shard to insert.                                               |
